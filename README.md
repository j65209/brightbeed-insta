# brightbeed-insta

인스타 릴스(9:16) 빠른 편집 도구. 이미지/영상 추가하고 텍스트 얹어서 미리보기까지 한 화면에서.

## 사용법

`index.html`을 브라우저로 열면 끝.

- **+ 이미지/영상 추가** — 좌측 큰 버튼으로 미디어 업로드
- 미디어 / 텍스트 모두 미리보기에서 **드래그로 위치 조정**
- 사이드바에서 크기·색상·굵기 편집
- `Delete` 선택된 텍스트 삭제, `Esc` 선택 해제

## 배포

Netlify에 연결되어 있음 (`netlify.toml` 참고). publish 디렉터리는 루트.
